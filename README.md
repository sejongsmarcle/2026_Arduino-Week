# 2026_Arduino-Week

>2026 SMARCLE Arduino Week

```text
2026 겨울방학 프로그램
1주일 동안 Arduino로 프로젝트를 만들고, 팀별로 결과를 공유하는 활동
```

## 진행 방식

### 1) 팀 객실
- 프로젝트 시 개인/팀 모두 참여 가능
- 팀은 자율 구성 가능

### 2) 활동
- 2/3(화) ~ 2/9(월)
- 2/3일 대면 OT와 1회차 수업 진행 및 실습
- 2/4, 2/5 비대면 TEAMS 2회차, 3회차 수업 진행
- 2/5 ~ 2/8 프로젝트 활동 대면/비대면 여부는 팀 내에서 조정됩니다.

### 3) 발표
규칙을 따른 후 최종발표 시간에 자신의 팀이 활동한 내용을 발표합니다. 
- 발표는 기본적으로 대면을 한다.
- 2/8(일) 자정(00:00)까지 본 깃허브에 프로젝트 자료 제출

## 일정
| 데이트 | 내용 |
|---|---|
| 2026년 1월 23일 ~ 2026년 1월 26일 | Arduino week 모집 |
| 2026년 2월 3일 | OT 및 1회차 수업 |
| 2026년 2월 4일 ~ 2026년 2월 5일 | 2회차, 3회차 비대면 수업 |
| 2026년 2월 6일~ 2월 8일 | 팀별 프로젝트|
| 2026년 2월 8일 00:00 | 제출 기간 |
| 2026년 2월 9일 15:00 | 최종 발표 |
